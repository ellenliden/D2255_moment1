Bootstrap

Bootstrap är ett populärt CSS-ramverk som används för att snabbt skapa responsiva och moderna webbplatser. Ramverket innehåller färdiga komponenter som knappar, formulär, tabeller och navigering.

Läs mer på Bootstraps webbplats: https://getbootstrap.com

Utvärdering av Bootstrap

Fördelar

  • Det är lätt att komma igång
  • Det finns bra dokumentation
  • Många färdiga komponenter
  • Responsivt direkt från start

Nackdelar

  • Ett populärt ramverk, vilket medför att många webbplatser får liknande utseende
  • Att ändra Bootstraps färdiga komponenter är lite krångligt och kan kräva mycket CSS
