Publicering

Vald publiceringsplattform

För att publicera denna webbplats har jag valt att använda Netlify. Netlify är en modern publiceringsplattform som är särskilt anpassad för statiska webbplatser och erbjuder ett enkelt arbetssätt utan att använda traditionell FTP.

Kodstöd

Netlify har stöd för statiska webbplatser byggda med bland annat HTML, CSS och JavaScript,

Plattformen har även stöd för olika JavaScript-ramverk som React, Vue och Svelte. I denna uppgift används dock endast HTML och CSS tillsammans med externa CSS-ramverk.

Kostnad

Netlify erbjuder en kostnadsfri plan som är fungerar bra för mindre projekt och uppgifter. Den fria planen erbjuder bland annat gratis hosting, automatiska publiceringar och SSL-certifikat.

Fördelar med Netlify

  • Gratis och enkel att använda
  • Ingen FTP krävs
  • Automatisk publicering via Git
  • Snabb och säker hosting

Arbetsflöde för publicering

Arbetsprocessen för att publicera webbplatsen ser ut enligt följande:

  1. Projektfilerna skapas och redigeras lokalt
  2. Filerna versionshanteras med Git
  3. Projektet laddas upp till ett GitHub-repository
  4. Netlify kopplas till repot
  5. Vid varje uppdatering i repot publiceras webbplatsen automatiskt

Detta arbetssätt gör det enkelt att uppdatera webbplatsen och det innebär att den publicerade versionen alltid är aktuell.
